# M2E Annotation Bundle

This repository contains the annotation web app and the current 180-trace sample used for collaborator review.

## Contents

- `APS_calculator/`
  - Local static annotation web app.
  - Open `APS_calculator/index.html` directly in a browser.
- `samples/recommended_sample_180.csv`
  - The current 180-record annotation sample.

## Quick Start

1. Open `APS_calculator/index.html` in a browser.
2. Import `samples/recommended_sample_180.csv`.
3. If you also have access to the original `runs` folder locally, connect it in the app to view the original traces during annotation.

No build step is required.
